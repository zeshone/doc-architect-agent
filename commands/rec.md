description: "Paso 1 — Elicitación de requerimientos para un sistema o módulo"
agent: doc-arch
---

Ejecuta el Paso 1 (elicitación de requerimientos) para el sistema o módulo indicado.

El usuario invocó: `/rec $ARGUMENTS`

Delega al sub-agente doc-rec con el argumento: $ARGUMENTS

El argumento puede ser:
- `<sistema>` → elicitación a nivel de sistema
- `<sistema>/<modulo>` → elicitación a nivel de módulo
- `<sistema>/<modulo>/<submodulo>` → elicitación a nivel de sub-módulo

Sigue el protocolo `rec` definido en tu skill para el nivel correspondiente.
